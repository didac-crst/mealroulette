# v0.7.0 — Cooking mode

Release date: 2026-07-12

## Summary

Phase 10 adds a **Today-first** mobile home, **cooking mode** for step-by-step recipes, **step timers** with browser chimes and optional Telegram alerts, and **real-time dish library search**. Existing planning, scheduler, shopping, and Telegram reminder behaviour is unchanged.

## Added

### Today home

- **`/today`** is the default landing page after login.
- Today's lunch and dinner cards with **Cook** and inline **Review** (eaten, skip, leftovers, rating).
- **View full week** links to the existing Review tab.

### Cooking mode

- Route **`/recipes/:recipeId/cook`** and **Cook** entry from recipe detail and Today.
- Step-by-step viewer with **Previous** / **Next**, collapsible full ingredient list, mobile-first layout.
- Optional **step timers** (minutes per step in recipe edit → countdown in cooking mode).
- Timers keep running when navigating between steps (**Running timers** bar).
- **Browser chime** (gentle repeating tone + vibration) and optional system notification when a timer finishes.
- **Dismiss** stops the alert and leaves the timer at **0:00** with **Reset** only.

### Telegram cooking timer alerts

- Starting a timer schedules a server-side alert; the worker delivers to all **subscribed** Telegram chats when the timer ends.
- Pause, reset, dismiss, and exit cooking mode cancel pending alerts.
- Migration `026_cooking_timer_alerts`.

### Dish library search

- Real-time client-side filter on dish name, description, and recipe variant names.

## Migration

- Alembic revision **`026_cooking_timer_alerts`** — `cooking_timer_alerts` table.

```bash
docker compose exec api alembic upgrade head
```

Restart **api** and **worker** after upgrading.


## Out of scope (deferred)

- Persistent cooking sessions (resume after refresh).
- Thermomix step layout, voice controls, Telegram deep links into cooking mode.
- Server-side dish search and tag/course filters (see [BACKLOG.md](../BACKLOG.md) PR #10 review follow-ups).
- Post-cook “mark eaten” shortcut from cooking mode.
